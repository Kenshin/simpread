含义
--

简悦可以同步配置文件到 **Dropbox** 或 **坚果云**。

更换保存源
---

> 选项页 → 高级设定 → 全局

默认保存到 **Dropbox**, 禁用后，保存到 **坚果云** 。

![2019-06-25_153712.png](https://i.loli.net/2019/06/25/5d11d00ec0bc761861.png)

使用
--
> 选项页 → 共通

![2019-06-25_154053.png](https://i.loli.net/2019/06/25/5d11d0a4d536692442.png)


注意事项
--

> 目前仅支持 `手动同步` 配置文件的功能；
