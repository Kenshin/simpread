恭喜，您的简悦已升级到最新 1.1.2 版本

简悦 的每一次 「进化」 都意味着，为您提供更优质的阅读体验。

本次升级为您准备了以下的新功能：

  • 丰富的插件系统
  • 更加强大的站点集市
  • 一目了然的待决站点机制
  • 账户系统
  • 重构了导出到 Evernote / 印象笔记 的服务
  • 全部功能

丰富的插件系统
丰富的插件系统：Lightbox · 划词搜索 · 划词翻译 更多插件等你发现！

1 现在开始可以使用 JavaScript 编写基于 简悦 的插件了，以及方便编写 插件编辑器，详细操作 请看这里；

2 以及为了方便查找插件的 插件中心；

3 在 FAP 增加了 插件 选项卡，方便对当前页面的插件进行管理（启/禁用）；

更加强大的站点集市
可查询全部的适配站点，包括：官方（主）适配源 · 第三方适配源（支持导入） · 站点集市适配源 · 自定义适配源；
除此以外，还支持用户自行上传适配站点，让你的适配站点服务数以万计的简悦用户。

1 站点集市；

2站点管理器 增加了上传机制，可以方便的上传自定义站点，如何上传请看 站点管理（上传）；

3 在 FAP 增加了 站点 选项卡，提供相同站的不同适配规则（通过点击 icon ）切换；

一目了然的待决站点机制
通过 待决站点 机制，解决了「如何方便提交站点」这件事情。
目前支持的场景包括：

1适配错误的站点；（适配失败时，会提示是否保存）

2临时阅读模式的站点；

3导出到 「Evernote / 印象笔记」失败的站点；（导出失败时，会提示是否保存）

注意：为了用户的隐私性，待决站点不会显示出用户信息。

账户系统
简悦终于有了自己账户系统，方便提交「站点 / 插件」时使用。
注意：1.1.2 版开放了 昵称 · 邮箱 · 联络方式 的修改，未来会有更多修改选项！

重构了导出到 Evernote / 印象笔记 的服务
1.1.2 解决了一直以来导出到 Evernote / 印象笔记 时 经常失败 的原因；
同时增加了当导出失败时，以 Markdown 格式 二次保存 及 提交到 待决站点机制，确保万无一失。
