简悦 SimpRead - 为你提供「如杂志般沉浸式阅读体验」的扩展 为了达到完美的阅读模式这个小目标 ，我适配了 数百种类型 的网站，因此诞生了简悦。 为了完美阅读模式这个小目标 ，我适配了 数百种类型 的网站，因此诞生了它 - 简悦 ：沉浸式阅读的扩展

大家好，我叫 Kenshin，简悦的作者。

主要功能包括：

  • 聚焦模式 / 阅读模式 / 临时阅读模式 / 主动阅读模式 / TXT 阅读器；
  • 站点编辑器，站点适配源，站点管理器；
  • 丰富的导出功能，包括：Pocket / Instapaper / Evernote / Onenote / Dropdox 等；
  • 多种字体样式、大小、版面、主题模式等；
  • 导出 / 导入 配置文件；
  • 自定义样式，详细请看 自定义主题；

阅读模式 可编程 支持：

    {
        "name"    : "qdaily.com",
        "url"     : "http://www.qdaily.com/articles/",
        "title"   : "<h2 class='title'>",
        "desc"    : "<p class='excerpt'>",
        "include" : "<div class='article-detail-bd'>",
        "exclude" : [
            "<div class='author-share'>",
            "<div class='embed-control'>"
            ]
    }
