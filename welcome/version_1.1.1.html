恭喜，您的简悦已升级到最新 1.1.1 版本

    简悦 的每一次 「进化」 都意味着，为您提供更优质的阅读体验。

    本次升级为您准备了以下的新功能：

    • 全新的控制栏面板
    • 更丰富的中文定制化
    • 黑名单
    • 无障碍阅读
    • 导出为 HTML 页面 与 临时阅读模式页面
    • 重新高亮与删除任意标签
    • 全部功能

    全新的控制栏面板

    阅读模式新增加了控制栏面板（Float Action Panel）

    1可以通过 选项页 → 高级设定 → 阅读模式 改为用传统的「浮动控制栏」；

    2未来会逐步替换为「控制栏面板」，所以不建议这么做；

    更丰富的中文定制化

    更丰富灵活的中文定制化，包括：行间距、字间距、无极调整的字体大小、版面布局等。

    1字体大小由原来的（大中小）三级 → 现在的任意调整；

    2版面布局由原来的（宽中窄）三级 → 现在的任意调整；

    3新增加一个字体类型：思源宋体（需本机安装对应字体）；

    4在控制面板就可以修改自定义字体样式（原来的方案是在自定义样式中修改）；

    5新增加了字间距、行间距、单词间距、首行缩进等选项（原来的方案是在自定义样式中修改）；

    黑名单

    有别于排除列表，加入到黑名单中的网页将不会载入简悦的代码，而前者仅仅是无法自动进入阅读模式；

    黑名单仅支持 URL 与 hostname，如：http://ksria.com/simpread/ | ksria.com | xxx.ksria.com

    无障碍阅读

    简悦早就应该加入此功能，希望能做些力所能及的事情。

    1得益于 Chrome 提供的 API；但可惜的是，Firfox 并未提供此 API；（Firefox 版不具有此功能）

    2支持快捷键盘 ts → 开始朗读 · tt → 停止朗读

    导出为 HTML 页面 与 临时阅读模式页面

    阅读模式的导出服务增加了两个新功能：导出当前页面为 HTML 到本地 · 临时阅读模式页面

    前者可以方便永久保存某些页面；后者可以生成一个临时的页面，方便一些无法使用的功能，如：有道云笔记等

    重新高亮与删除任意标签

    重新高亮：使用后会重新出现选取高亮的框选框（类似与临时阅读模式）一般用于：当智能识别模式识别有误时的场景。
    删除任意元素：可以删除阅读模式下任意内容，与排除列表的区别是：前者只是临时删除，刷新页面后仍可出现。
