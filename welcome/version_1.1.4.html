恭喜，您的简悦已升级到最新 1.1.4 版本

    简悦 的每一次 「进化」 都意味着，为您提供更优质的阅读体验。

    本次升级为您准备了以下的新功能：

    • 导出服务已经支持 有道云笔记 · 为知笔记 · Notion · Bear · Ulysses
    • WebDAV 定制化导出文件的类型
    • 原生的离线下载
    • 更友好的反馈渠道
    • 更方便的「黑名单 · 白名单 · 排除列表 · 延迟列表」编辑器
    • 词法分析引擎
    • 已失效的插件
    • 全部功能

    导出服务已经支持「有道云笔记 · 为知笔记 · Notion · Bear · Ulysses」

    真的久等了！简悦终于支持了 有道云笔记 · 为知笔记 · Notion
    注意：由于 有道云笔记 · 为知笔记 · Notion 并未提供 API 所以它们的授权方式各不一样，同时也存在小概率的授权失败情况。 😅
    小提示：Mac OS 用户，简悦也可以导出到 Bear · Ulysses 了

    WebDAV 定制化导出文件的类型

    导出到 WebDAV 的格式，除了 默认的 MD 外，先增加了 HTML 以及 离线 HTML 说明文档

    原生的离线下载来了

    插件中心原本有一个 离线下载 但在处理多图的时候，性能并不理想，所以新版本原生的增加了 离线下载
    使用方法：

    1阅读模式 → 控制栏 → 动作 → 导出

    2离线下载需要新的权限，请确认给予

    更友好的反馈渠道

    原本简悦的反馈渠道只有一个 Github Issues
    为了方便无 Github 帐号的朋友，增加了一个新的反馈渠道 腾讯 · 吐个槽
    反馈位置有两个：

    1阅读模式下方便的页脚

    2选项页右下角

    简单的说下「吐个槽」

    1无需注册，点击后会自动使用简悦的注册系统

    2如需实时收到反馈，请根据提示关注（腾讯官方）微信号

    3用户的提交内容，他人无法看到

    墙裂注意：还是希望可以通过 Github Issues 反馈！
    同时，新增一个投票效果，希望你能喜欢

    更方便的「黑名单 · 白名单 · 排除列表 · 延迟列表」编辑器

    将当前页面发送到「黑名单 · 白名单 · 排除列表 · 延迟列表」时可进行二次编辑，提供此项功能的效率。
    同时支持：

    1主域名

    2Minimatch

    2域名

    2正则表达式

    除此以外，延迟加载也加入到右键菜单

    升级了词法分析引擎

    新的词法分析引擎，包含了以下三个功能：
    代码段可识别更多类型

    1crayon-syntax e.g. site

    2syntaxhighlightere.g. site

    3table e.g. site

    阅读模式可识别特殊结构
    适配站点增加 URL 的正则表达式支持
